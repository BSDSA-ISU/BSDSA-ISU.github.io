About this course

BSDSA is a course within the CCSICT department that focuses on analyzing, interpreting, and visualizing data.


So what does this course offer?

  • Programming, coding, and logic development
  • Database knowledge
  • Python programming
  • Ethics

Who are we?

We are the students of BSDSA ISU currently located in San Fabian, Echague, Isabela.


🚀 Our Goals

  • Build strong foundations in data science, analytics, and programming
  • Collaborate on real-world data projects
  • Develop critical thinking and problem-solving skills
  • Stay curious about emerging technologies
  • Support each other as a learning community
  • Prepare for careers in tech, research, and innovation
  • Survive the course (optional but realistic)

📚 What we learn here

  • Python, R, and SQL programming
  • Machine learning and data pipelines
  • Cloud platforms and database systems
  • Data visualization and storytelling
